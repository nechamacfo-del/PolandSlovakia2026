# אתר הטיול לפולין – משפחת דורי

## פתיחה במחשב
1. חלצי את קובץ ה-ZIP.
2. פתחי את הקובץ `index.html` בדפדפן.

## העלאה ל-GitHub Pages
1. פתחי חשבון ב-GitHub.
2. צרי Repository חדש בשם, לדוגמה: `Poland2026`.
3. העלי אליו את שלושת הקבצים:
   - index.html
   - style.css
   - app.js
4. היכנסי ל-Settings > Pages.
5. תחת Build and deployment בחרי:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. לחצי Save.
7. לאחר כמה דקות יתקבל קישור בסגנון:
   https://YOUR-USERNAME.github.io/Poland2026/

## עריכת המסלול
כל הימים נמצאים בתחילת הקובץ `app.js`, בתוך המשתנה `days`.
אפשר לערוך שם:
- תאריך
- כותרת
- תקציר
- פרטי היום
- קישור ניווט

זהו אבטיפוס ראשוני. אין בו פרטים אישיים, מספרי הזמנות או מסמכים.
